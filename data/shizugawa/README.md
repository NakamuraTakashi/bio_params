## 単位・仕様メモ

GLODAP と統一すべき項目（loader 側で吸収する）:

- **栄養塩の単位**: GLODAP は µmol/kg。ローカルが µmol/L の場合、海水密度で換算する
- **Chl-a の単位**: 統一すること（µg/L 等）
- **欠損値の表現**: ローカル仕様を確認し、loader で NaN に変換
- **品質フラグ**: 自前の品質基準を WOCE 風（good/補間/欠損）に整理し、`<target>_flag` 列に統一
- **座標系**: 緯度経度・水深（メートル単位）

## 注意

- 内湾は外洋と T-S–生物地球化学関係が質的に異なる（河川流入・浅水・季節変動が大きい）
- GLODAP の塩分分布（外洋 ~33–37）外に出る低塩分データは、事前学習モデルの外挿になる
- 詳細な背景は `CLAUDE.md` の「ローカルデータの統合」セクション参照
